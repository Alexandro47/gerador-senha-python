# 🔐 Gerador de Senha em Python

Meu primeiro projeto em Python feito no VSCode! Um gerador de senha 100% customizável feito do zero.

### ✨ Funcionalidades:
- Define tamanho da senha 
- Escolhe: letras, números e símbolos
- Valida se usuário digitou número com try/except
- Avisa se a senha é forte ou fraca
- Salva todas senhas geradas em senhas.txt

### 🚀 Como usar:
```bash
python gerador.py
```
### O que aprendi:
`import`,`input`,`while`,`try/expect`,`for`,`if`,`else`,`indentação`,`manipulação de arquivos`
